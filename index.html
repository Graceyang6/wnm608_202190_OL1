Create a system: basic + variants (warning, alert, money, hero).

Usage Notes

  • Primary: Add to cart, Checkout, Confirm
  • Money: Purchase / price-related CTA
  • Warning: “Are you sure?” actions
  • Alert: Destructive actions (remove, delete)
  • Hero: One main CTA in a marketing section
System rule: Every button must have hover + focus + disabled states.
